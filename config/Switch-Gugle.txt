Hostname: Switch-Gugle
Platform: Cisco Catalyst (Simulated - PT)
OS       : IOS 15.0

------------------------------------------------------------
 ROLE & TOPOLOGY
------------------------------------------------------------
- Menjadi access switch untuk:
  * Web Server  (172.168.0.0/24)
  * DNS Server  (172.168.0.0/24)
- Terkoneksi ke Router-Gugle melalui port GigabitEthernet0/1.
- Seluruh perangkat server berada di satu subnet tanpa VLAN
  tambahan (menggunakan default VLAN 1).

------------------------------------------------------------
 INTERFACE CONFIGURATION
------------------------------------------------------------

# Semua port menggunakan VLAN default (VLAN 1)
# Port terpakai:
#   - Fa0/1 : Web Server
#   - Fa0/2 : DNS Server
#   - Gi0/1 : Uplink ke Router-Gugle

 ! Connected to Web Server (Fa0)

interface FastEthernet0/2
 ! Connected to DNS Server (Fa0)
