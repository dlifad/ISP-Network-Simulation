 IGP CONFIGURATION (OSPF)
------------------------------------------------------------

router ospf 300

 # Jaringan server Gugle
 network 11.11.11.0 0.0.0.3 area 0

 # Redistribusi route dari BGP ke OSPF
 redistribute bgp 300 subnets

------------------------------------------------------------
 EGP CONFIGURATION (BGP)
------------------------------------------------------------

 no synchronization

 # Peering ke RISP-Lain (AS 200)
 neighbor 30.30.30.1 remote-as 200

 # Redistribusi OSPF ke dalam BGP
 redistribute ospf 300
